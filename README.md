Kartooni – Cartoon Episodes Web Page

This project is a simple web page designed to display cartoon episodes in a clean and visually appealing layout using HTML and CSS.

📄 Project Description
A main page that showcases cartoon episodes using card-style components.
Includes a header with the website title and image.
A main section displaying episode cards with images, icons, and episode numbers.
Interactive buttons for joining a Telegram channel and watching more series.
A footer containing general information and copyright.
🎨 Technologies Used
HTML5
CSS3
🖼️ Project Structure
index.html – Main web page
css/index.css – Stylesheet for design and layout
image/ – Folder containing images:
header.jpg – Website header image
nav.jpg – Navigation/cover image
card1.jpg, card2.jpg – Episode images
icon.jpg – Card icon
⚡ Features
Fully supports RTL (Right-to-Left) layout for Arabic content
Card-based design for displaying episodes
Interactive buttons for external actions
Clean and simple footer section
🛠️ How to Use
Download or clone the project files.
Open index.html in your browser.
Make sure the css and image folders are in the correct paths for proper styling and images.
📌 Notes
This is a demo project for practice purposes.
The website does not store or host any files on a server.
You can easily customize images and content as needed.
